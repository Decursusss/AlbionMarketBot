## Getting Started
 
### Step 1 Download the Bot
 
Download the latest release:
 
👉 **[Download MarketBot.zip](https://github.com/Decursusss/AlbionMarketBot/releases/download/V1.0/MarkeBot.rar)**
 
Extract the ZIP to any folder on your PC.
 
---
 
### Step 2 Get Your License Key
 
The bot requires a license key to run.
 
1. Open Telegram and find the bot: **[@bastrikbot](https://t.me/bastrikbot)**
2. Send the command `/start`
3. Follow the instructions to get your key
4. Copy the key — you'll need it in the next step
---
 
### Step 3 Activate the Bot
 
1. Open the extracted folder
2. Run **`MarketBot.exe`**
3. When prompted, paste your activate key
4. Click **Login**
